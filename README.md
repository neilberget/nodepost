# Nodepost

Query Nodes in a Nodegroup

## Installation

Add this line to your application's Gemfile:

    gem 'nodepost'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install nodepost

## Usage

Command line: 

First, make sure the NODEGROUPS\_BASE\_URL environment variable is set, then:

    nodepost <nodegroup>
